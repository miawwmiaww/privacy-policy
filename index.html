Kebijakan Privasi

Terakhir diperbarui: 30 Desember 2025

Kami menghargai privasi Anda. Halaman ini menjelaskan bagaimana bot Telegram kami mengumpulkan, menggunakan, dan melindungi data pengguna dalam layanan Closed Testing.

1. Informasi yang Dikumpulkan

  • Nama aplikasi dan package
  • Email tester
  • ID pesanan dan status
  • Telegram User ID

2. Penggunaan Data

Data digunakan untuk memproses layanan Closed Testing, pembayaran, dan pengiriman status.

3. Pembayaran

Pembayaran diproses melalui pihak ketiga (Midtrans / QRIS). Kami tidak menyimpan data pembayaran sensitif.

4. Pihak Ketiga

Data hanya dibagikan kepada layanan terkait seperti Google Play dan penyedia pembayaran.

5. Keamanan Data

Kami menjaga keamanan data dengan langkah wajar sesuai praktik terbaik.

6. Hak Pengguna

Anda dapat meminta penghapusan atau koreksi data dengan menghubungi admin bot.

7. Perubahan

Kebijakan ini dapat diperbarui sewaktu-waktu.

8. Kontak

Hubungi admin melalui Telegram Bot ini. atau melalu halaman website metronewtech.site
